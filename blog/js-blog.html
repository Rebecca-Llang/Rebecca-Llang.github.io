JS, HTML & CSS

HTML is the skeleton of your webpage. It provides the basic structure, like headers, text, images, and links. Think of HTML as the backbone and the various elements as the limbs that extend and interact with the CSS and JS. Without HTML, the website would lack structure and content.

CSS is the aesthetics of your webpage. CSS is responsible for how everything looks and is positioned on the screen. Imagine CSS as the clothing and style that make the HTML look appealing. It tells the web browser how to display HTML elements, controls things like colors, fonts, and layouts. CSS brings beauty and elegance to the structured skeleton framework provided by HTML. It is the interior decorator of the web.

JavaScript is the brain, dictating action or behavior of your webpage. JS makes your website interactive and responsive to user actions. For example, JavaScript can handle what happens when you click a button, type in a form, or upload a file. It's like the brain of the operation, directing actions and making decisions to enhance user experience.

JS RELATION TO HTML & CSS

As detailed above, JavaScript allows us to change and edit the content and style of our webpage. JS often uses objects, functions and loops to make what you want to happen, happen. It references classes and ID’s to interact directly with what it needs to. Classes are often used for CSS, and ID’s for HTML elements. JS allows us to build complex, dynamic projects. This allows us to nest, build and offer many options, solutions and next steps when interacted with by the user, as well as a way to edit our DOM quickly in the future.

JS interacts with HTML and CSS, within the DOM to edit, read and write to the files that build our overall finished project.

CONTROL FLOW & LOOPS

JavaScript’s control flow is built by using statements and functions such as if/else and switch statements, loops and function calls.

If statements, will check if a parameter is met, and depending on if it is true, do a certain function. You can use if/else statements to provide other options of what to do if the initial parameters are false. Switch statements use a similar system, however the syntax is different and provides a ‘case’ of each option of the action in reference to the outcome of conditions of the parameter.

Loops allow us to engage with arrays and objects repeatedly, using iteration. Parameters set the initial value of the iteration, how many times to iterate through the object/array and what to do after the initial loop.

Use an If statement if you want to perform a certain action if the parameters are met. Use if/else to provide other actions under different conditions.

Use loops to repeat tasks, iterate over arrays and objects, do something until certain conditions are met, to do calculations and manipulate or search data.

ARRAYS AND OBJECTS

Arrays are containers that hold a collection of elements with the same type of data. Arrays start their element position at 0. An array could have a .length of ‘4’, which would be placed as [0,1,2,3] within the array. We can access elements in the array using their position in the index. They are used when you want to group similar items, such as names, products or numbers.

Objects are containers that are used to group multiple different sets of related data. Each set of data is given in pairs, with a key and a value. They are used for things like individual people’s information, and as a reference for functions and different sections of your DOM. They can also allow us to access properties such as other objects and functions.

Think of arrays as a way to store and access data.

We can see objects as a way to structure data, with properties that describe what the section is, and the functions we want to perform within them.
